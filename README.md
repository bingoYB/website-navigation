目录结构
---

```
├── README.md
├── package.json
├── docs                  # 打包目录 实际页面访问目录
├── src                   # Vue.js 页面所有代码
│   ├── data              # 数据文件
│   ├── img               # 图片资源
│   ├── lib               # 组件依赖
│   ├── page              # 页面代码
│   ├── script            # js脚本
│   ├── style             # 样式字体等
│   ├── template          # 模版 tpl
│   ├── index.html        # 页面入口文件
```
